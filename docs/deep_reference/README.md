# Deep Reference Learning Hub

This folder contains in-depth week-aligned reference material for the AI/ML curriculum.

## How to use
- Read the weekly deep dive before/after the corresponding hands-on notebook.
- Use sections: concepts, workflow, tools, references, challenge.
- Cross-reference with `guides/` and the phase starter notebook folders under `phases/` for practical implementation.
